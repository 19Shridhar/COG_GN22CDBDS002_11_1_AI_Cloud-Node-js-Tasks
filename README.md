# COG_GN22CDBDS002_11_1_AI_Cloud-Node-js-Tasks

Node Js Web app which perform NLP string operations

File Description:-

1) index.js contains the main app which renders the html input page and also processes the post request of the input. The output is send back using res.write()

2) src\utils\func.js this file contains all the necessary functions to perform the following tasks.

    1. Sentence Segmentation
    2. Word Segmentation
    3. Sentence without Stop Words
    4. Sentence without repeated strings
    5. Sentence with each word of the string reversed
    6. all numbers extracted from the string

3) public\index.html is the html document on the homepage which contains the input form
